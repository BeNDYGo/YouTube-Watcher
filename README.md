Расширение для YouTube, которое позволяет сохранять видео во временный плейлист прямо на странице без открытия новых вкладок.

>Просто зажмите "a" при клике на видео и ролик добавится во временный плейлист

⭐*буду благодарен за звездочки*⭐

![Demo](https://raw.githubusercontent.com/BeNDYGo/YouTube-Watcher/refs/heads/main/output.gif)

- Быстрое восстановление плейлиста

# Установка

1. Скачайте последнюю версию из Releases
2. Откройте chrome://extensions/
3. Включите "Режим разработчика"
4. Нажмите "Загрузить распакованное расширение"
5. Выберите папку проекта

# Работа:

на страницу добавляется временный плейлист, в который можно добавить ролик, нажав на видео с зажатой кнопкой "a"(русская/английская) 

Окно расширения можно перемещать в разные углы по странице.

А так же вы можете менять отображаемое количество видео, тем самым меняя размеры, занимаемые окном расширения

Если нажать на иконку расширения откроется меню, где можно посмотреть актуальноть версии и **открыть заново плейлист, если вы его закрыли**
